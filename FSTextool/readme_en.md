# FSTextool

[sc](https://github.com/mkitto/fsll/blob/development/FSTextool/readme.md) | en <br>

A c# clone of falloutshelter_texttool (au3(c++) by swuforce). <br>
The input and output files are the same as the swuforce version. <br>
This tool used to unpack and repack specific dat file of Fallout Shelter pc. Based on tool by swuforce[<sub>1</sub>](https://zenhax.com/viewtopic.php?f=12&t=2770),[<sub>2</sub>](https://zenhax.com/viewtopic.php?t=645), [LeHieu](https://github.com/lehieugch68/UE4-Locres-Unpacker), Qé…±.

## Usage

Drag and drop Dat/Txt files into this tool to unpack / repack.
